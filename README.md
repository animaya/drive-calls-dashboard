# Дашборд анализа качества звонков

Интерактивный дашборд для анализа качества звонков менеджеров автошколы "Драйв".

## Возможности

- Общий обзор по всем менеджерам
- Детальный анализ по 14 критериям
- Цветовое кодирование результатов
- Модальные окна с подробной информацией

## Демо

Откройте [GitHub Pages](https://username.github.io/drive-calls-dashboard/) для просмотра дашборда.

## Локальный запуск

```bash
python3 -m http.server 8000
```

Откройте http://localhost:8000 в браузере.

## Структура

- `index.html` - основной файл дашборда
- `results/` - JSON файлы с результатами анализа
  - `aggregated_results.json` - сводные данные по всем менеджерам
  - `*_analysis_full.json` - детальные данные по каждому менеджеру
